# Routing Rules

## Main Rule
Routing is based on:
- recruitment branch
- level classification or routing basis

## Branch Rules
- Plantilla uses stricter branch-aware workflow rules
- COS uses lighter flexible workflow rules

## Level Rules
- Level 1 cases are assigned to the Secretariat
- Level 2 cases are assigned to the HRM Chief
- Secretariat must be blocked from processing Level 2 cases

## COS Rule
- The same Level 1 / Level 2 routing logic may be applied to COS as an internal office control

## Override Rule
- Routing override is not automatic
- If implemented, it must be:
  - authorized
  - controlled
  - audit-logged
