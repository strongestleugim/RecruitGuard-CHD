Official Recruitment Application Portal

This portal is used for public submission of applications to current DOH-CHD CALABARZON recruitment entries. Applicants may proceed through the appropriate Plantilla or Contract of Service path, complete OTP verification, and use the issued Application ID to check submission status.

Start New Application Check Status

Portal Purpose

Submit applications only for currently advertised recruitment entries. Review the correct branch and entry details before proceeding.

Before Submission

Prepare all required information and supporting documents. Final submission is accepted only after successful OTP verification.

After Submission

Keep the generated Application ID. Use the Status Lookup page to view the latest recorded submission state.

Available Openings and Application Paths

Select the correct recruitment path before applying. Plantilla and Contract of Service entries are maintained separately and may have different intake conditions.
